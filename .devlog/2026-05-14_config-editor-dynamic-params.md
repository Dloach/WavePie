# 2026-05-14: 配置编辑器重做 — 动态参数面板 + 键盘录制 + 宏/脚本编辑器

## 操作人
AI

## 涉及版本
client-v1

## 做了什么

### 1. 去掉无效的图标字段
从菜单项编辑中去掉了 emoji 图标输入框（无选择器，形同虚设）。

### 2. 参数面板根据命令类型动态切换
每行菜单项的右侧参数区现在根据 `action_type` 变化：

| 类型 | 参数控件 |
|------|---------|
| `key_combo` | 文本框（手动输入组合键）+ 🎬 录制按钮 |
| `macro`     | 编辑按钮 → 弹出多行文本编辑器弹窗 |
| `script`    | 浏览按钮 → Windows 文件选择器（.bat/.ps1/.py）|
| `log`       | 普通文本输入框 |

### 3. 键盘录制功能
点击「🎬 录制」→ 弹出模态窗口 → pynput 监听按键：
- 自动识别修饰键（Ctrl/Alt/Shift/Win）
- 支持字母键、数字键、功能键（F1-F12）、方向键等
- 按下主键后自动停止录制
- 确认后保存为标准格式（如 `ctrl+z`、`win+shift+s`）
- 取消则放弃

### 4. 多行宏编辑弹窗
选择 macro 类型 → 点击「编辑」→ 弹出独立窗口
- 深色主题多行文本区
- 支持粘贴/直接编写

### 5. 脚本文件选择器
选择 script 类型 → 点击「浏览…」→ 系统文件对话框
- 过滤 .bat / .ps1 / .py / .vbs / .cmd
- 保存完整路径

### 6. 同样的动态面板用于
- 菜单项编辑
- 副键直接动作编辑
- 滚轮上/下映射编辑
