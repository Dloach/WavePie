### 关键 Bug 修复

1. **多显示器不显示菜单**（KB-001）
   - 根因：GetMonitorInfoW 返回屏幕坐标，Canvas 用本地坐标，虚拟桌面原点非 (0,0) 时差了一整个偏移量
   - 修复：`canvas_x = cx_screen - vx`

2. **准星重影**
   - 根因：`_draw_sight` 创建十字线后未删除，每帧叠加
   - 修复：追踪 `_crosshair_ids`，每帧清除

3. **BLE 断开后不广播**（KB-004）
   - 修复：ServerCB::onDisconnect 加 `s->startAdvertising()`

4. **菜单编辑器 UI**
   - 双击托盘打开两个 → 500ms 防抖
   - 参数框宽度互相影响 → 固定宽度 ActionParamFrame
   - 类型切换不跟新参数 → ActionParamFrame.switch_type()
   - 窗口不可调 → resizable(True, True)

5. **固件**
   - Madgwick 滤波 → 陀螺仪 Z 轴积分（无磁力计 heading 不可靠）
   - 按键去抖 15ms → 边缘检测
   - BLE 跨核竞态 → FreeRTOS xQueue 解耦
   - 所有 Serial 输出删除 → BLE 速率稳定 250Hz
